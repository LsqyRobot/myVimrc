# 📚 文档目录

这里包含了所有的详细文档，按用途分类：

## 🚀 快速开始

### [AUTO_INSTALL_GUIDE.md](AUTO_INSTALL_GUIDE.md) 🚀 **一键安装**
- **用途**: 自动安装模式详细指南
- **特点**: `./build.sh --auto` 零交互安装
- **内容**: 使用方法、功能对比、常见问题
- **适合**: 想要快速体验配置的用户

### [SIMPLE_INSTALL_GUIDE.md](SIMPLE_INSTALL_GUIDE.md) ⭐ **推荐新手**
- **用途**: 简化版安装和使用指南
- **特点**: 专注 C/C++/Python 三种语言
- **内容**: 修复弃用警告、核心功能、快捷键说明
- **适合**: 新用户或只需要基础功能的用户

### [MACOS_INSTALL_GUIDE.md](MACOS_INSTALL_GUIDE.md) 🍎 **Mac 用户专用**
- **用途**: macOS 系统专用安装指南
- **特点**: Homebrew 自动配置、终端字体设置
- **内容**: 系统集成、性能优化、故障排除
- **适合**: 所有 macOS 用户 (Intel 和 Apple Silicon)

## 📖 详细指南

### [NEOVIM_GUIDE.md](NEOVIM_GUIDE.md)
- **用途**: 完整的 Neovim 使用指南
- **特点**: 全功能详解，包含所有插件说明
- **内容**: 详细快捷键表、插件配置、故障排除
- **适合**: 希望深入了解所有功能的高级用户

### [UPGRADE_GUIDE.md](UPGRADE_GUIDE.md)
- **用途**: 版本升级和配置迁移指南
- **特点**: 升级步骤详解
- **内容**: 版本升级、配置迁移、兼容性说明
- **适合**: 需要升级现有配置的用户

## 🛠️ 故障排除

### [INSTALL_FIX_GUIDE.md](INSTALL_FIX_GUIDE.md)
- **用途**: 安装问题修复指南
- **特点**: 解决插件安装和配置问题
- **内容**: 插件错误、安全检查、智能降级
- **适合**: 遇到安装问题的用户

### [FILETREE_ICONS_FIX.md](FILETREE_ICONS_FIX.md) 🔧 **图标问题**
- **用途**: 文件树图标乱码问题解决
- **特点**: 多种解决方案，从简单到完整
- **内容**: 简单图标、字体安装、故障排除
- **适合**: 文件树显示乱码的用户

## 🗂️ 文档使用建议

### 对于新用户
1. **macOS 用户**: 先阅读 [MACOS_INSTALL_GUIDE.md](MACOS_INSTALL_GUIDE.md) 🍎
2. **Linux 用户**: 先阅读 [SIMPLE_INSTALL_GUIDE.md](SIMPLE_INSTALL_GUIDE.md)
3. 完成安装并测试基本功能
4. 需要更多功能时参考 [NEOVIM_GUIDE.md](NEOVIM_GUIDE.md)

### 对于现有用户
1. 升级时参考 [UPGRADE_GUIDE.md](UPGRADE_GUIDE.md)
2. 遇到问题时查看对应指南的故障排除部分

### 对于高级用户
- 直接参考 [NEOVIM_GUIDE.md](NEOVIM_GUIDE.md) 进行自定义配置
